Overview

Threads Downloader is a browser extension that allows users to download media from Threads.net. This Privacy Policy explains how we handle your data when you use our extension.

By installing and using Threads Downloader, you agree to the practices described in this policy. If you do not agree with these practices, please do not use the extension.

What Data We Collect

Data You Provide Directly

  • Threads usernames (if provided for queue file processing)
  • Queue files containing download lists
  • Any manual input through the extension interface

Data Collected Automatically

  • Extension settings and preferences (stored locally)
  • Download history (stored locally on your device)
  • Queue file contents for batch processing

Data Processed During Use

  • Media URLs from threads.net pages you visit
  • Page content necessary for media extraction
  • Download status and progress information

Data We Do NOT Collect

  • Threads account credentials or login information
  • Browsing history outside of threads.net
  • Personally identifiable information (PII)
  • Any data transmitted to external servers

How We Use Your Data

The data processed by Threads Downloader is used solely for the following purposes:

  1. Media Detection: Extracting media URLs from Threads.net pages to enable download functionality
  2. Download Management: Processing and initiating downloads as requested by the user
  3. Queue Processing: Handling batch download operations from queue files
  4. Local Storage: Saving user preferences and download history on your device for improved user experience

How Data Is Shared

We do not share, sell, or transfer any user data to third parties.

All data processing occurs locally on your device. We do not have access to your data and do not transmit any information to external servers except for the necessary media downloads from threads.net that you explicitly initiate.

Security Measures

Local Storage Only

All data is stored locally on your device using the browser's local storage API. We do not use external databases or cloud storage.

Input Validation

All user input is validated before processing to prevent potential security issues.

Domain Restrictions

The extension only operates on threads.net domains and does not access or modify data on other websites.

No External Transmission

Except for downloading media from threads.net that you explicitly request, no data is transmitted to external servers.

Your Rights Regarding Data

You have full control over your data when using Threads Downloader:

Access Your Data

All your data is stored locally on your device. You can access it by:

  1. Opening Chrome/Edge extensions page
  2. Clicking on "service worker" or "background page" for Threads Downloader
  3. Using the Application/Storage tab in developer tools

Delete Your Data

To delete all stored data:

  1. Go to chrome://extensions
  2. Find Threads Downloader and click "Remove"
  3. Alternatively, clear site data for threads.net in your browser settings

Control Extension Behavior

You can disable or uninstall the extension at any time to stop all data processing.

Export Your Data

Your download history and settings are stored locally. You can export this data through the browser's developer tools if needed.

Data Retention

Data is retained only as long as you keep the extension installed:

  • Settings & Preferences: Retained until extension is removed
  • Download History: Retained until extension is removed or history is cleared
  • Queue Files: Retained until processing is complete or file is deleted

Children's Privacy

Threads Downloader is not intended for use by children under 13 years of age. We do not knowingly collect data from children. If you believe we have inadvertently collected data from a child, please contact us immediately.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time to reflect changes in our practices or for legal reasons. If we make material changes, we will:

  • Update the "Last Updated" date at the top of this policy
  • Post the updated policy in the extension and on this page

We encourage you to review this Privacy Policy periodically.

Contact Information

If you have questions, concerns, or feedback about this Privacy Policy or our data practices, please:

  • Open an issue on our GitHub Issues page
  • Visit our GitHub Repository
