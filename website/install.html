Installation Guide

Get started with Threads Downloader in minutes

Chrome & Chromium Browsers

Recommended installation method

1

Open Chrome Web Store

Click the button below to open the Threads Downloader page in the Chrome Web Store.

2

Click "Add to Chrome"

Find the blue "Add to Chrome" button in the top right corner of the page.

3

Confirm Installation

A dialog will appear asking for permission. Click "Add extension" to confirm.

4

Pin the Extension

Click the puzzle piece icon in your browser toolbar and pin Threads Downloader for easy access.

Open Chrome Web Store

Firefox

Install from Firefox Add-ons

1

Open Firefox Add-ons

Click the button below to open the Threads Downloader page on Firefox Add-ons.

2

Click "Add to Firefox"

Find the blue "Add to Firefox" button on the add-on page.

3

Grant Permissions

A dialog will appear asking for permission to add the extension. Click "Add" to confirm.

4

Pin the Extension

Click the puzzle piece icon in your browser toolbar and pin Threads Downloader.

Open Firefox Add-ons

Manual Installation

For development or testing purposes

1

Download the Source

Download the source code from GitHub Releases or clone the repository.

2

Enable Developer Mode

Open chrome://extensions in Chrome or about:debugging in Firefox and enable Developer Mode.

3

Load Unpacked Extension

Click "Load unpacked" (Chrome) or "Load Temporary Add-on" (Firefox) and select the extension directory.

4

Pin and Use

Pin the extension to your browser toolbar. The extension will be temporary in Firefox.

How to Use

Start downloading media in 3 simple steps

1

Navigate to Threads

Visit threads.net and find a post with media you want to download.

2

Click the Extension Icon

Click the Threads Downloader icon in your browser toolbar. The popup will show detected media.

3

Download Media

Click "Download" next to any media item, or "Download All" to save everything.

Troubleshooting

Common installation issues and solutions

Extension Not Appearing

Make sure Developer Mode is enabled. Try refreshing the extensions page after installation.

Popup Not Working

Check that the extension has permission to access threads.net. Try pinning the extension to the toolbar.

Downloads Not Starting

Ensure the extension has permission to download files. Check your browser's download settings.
