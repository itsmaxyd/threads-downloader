Version Guide

Understanding our versioning

Major Version (1.x.x)

Significant new features, breaking changes, or major redesigns. Updates that fundamentally change the extension's behavior.

Minor Version (x.1.x)

New features, improvements, and backwards-compatible changes. Minor version increments indicate feature additions.

Patch Version (x.x.3)

Bug fixes, security updates, and minor improvements. Patch versions are backwards-compatible fixes.
